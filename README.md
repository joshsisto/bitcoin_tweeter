# bitcoin_tweeter
Python 3.4.2 | tweepy 3.5.0

## Description
Twitter bot written in Python 3 using the tweepy library that tweets the current bitcoin price and the price of bitcoin 1 year ago.

## Installation
Install Tweepy
    pip3 install tweepy
    
Add to cron
